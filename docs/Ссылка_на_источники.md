Основные **онлайн-редакторы и демо Blockly**, где можно сразу попробовать и посмотреть, как работает Blockly (drag-and-drop блоки, генерация кода и т.д.):

| №  | Название / Описание                              | Ссылка                                                                 | Что там можно делать                              |
|----|--------------------------------------------------|------------------------------------------------------------------------|---------------------------------------------------|
| 1  | Официальный Blockly Playground (самый мощный)   | https://blockly-demo.appspot.com/static/tests/playground.html        | Полный тестовый редактор, все блоки, настройки   |
| 2  | Демо Blockly → Code (классика, блоки → код)     | https://blockly-demo.appspot.com/static/demos/code/index.html        | Простой редактор + генерация JS/Python и др.     |
| 3  | Blockly Games (обучающие игры на Blockly)       | https://blockly.games/                                                    | Уроки через игры (Puzzle, Maze, Turtle и т.д.)   |
| 4  | JDoodle — онлайн Blockly IDE                     | https://www.jdoodle.com/blockly-online                                    | Пишешь блоки, запускаешь код онлайн              |
| 5  | Blockly Developer Tools (старый Block Factory)  | https://blockly-demo.appspot.com/static/demos/blockfactory/index.html | Создание своих блоков (для продвинутых)          |
| 6  | Blockly IO24 Demo (простая интерактивная демка) | https://developers.google.com/blockly/io24-demo                           | Быстрый "Hello World" и эксперименты             |
| 7  | Русскоязычный Blockly.Ru (среда + примеры)      | https://blockly.ru/ или https://blockly.ru/code/index.html               | Русский интерфейс, примеры, черепашка            |
| 8  | Codesandbox с Blockly (можно форкать)           | https://codesandbox.io/s/blockly-2hljy                                    | Онлайн-песочница для экспериментов с кодом       |

**Самые популярные для быстрого старта**:
- https://blockly-demo.appspot.com/static/tests/playground.html — здесь почти всё, что есть в Blockly
- https://blockly-demo.appspot.com/static/demos/code/index.html — классический пример с выводом кода
