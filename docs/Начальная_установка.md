## Начальная установка

### Шаг 1: Обновление системы и установка базовых пакетов
```bash
sudo apt update
sudo apt upgrade -y
sudo apt install -y curl git build-essential ca-certificates gnupg
```

### Шаг 2: Установка Node.js 22.x LTS (рекомендуемая версия)
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt update
sudo apt install -y nodejs

# Проверка версий
node -v  # должно показать версию ≥ 22.x
npm -v
```

### Шаг 3: Создание Vue.js проекта с TypeScript
```bash
npm create vue@latest
```

При создании проекта:
- На вопрос "Ok to proceed? (y)" отвечаем `y`
- Выбор опций производим с помощью пробела
- Подтверждение выбора клавишей Enter

### Шаг 4: Установка Blockly
```bash
npm install blockly
# Или для установки последней версии:
npm install blockly@latest
```

### Шаг 5: Проверка установки Blockly
```bash
npm list blockly
```

### Шаг 6: Запуск проекта
```bash
npm run dev -- --host
```
